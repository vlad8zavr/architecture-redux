# architecture-redux

**Структура проекта**

1. страница - `./src/index.html`
2. файл сборки js - `./build/reduxFlow.js`
3. точка входа js - `./src/index.js`

4. Папка **src**

    a) `./src/store`
    b) `./src/view`
    c) `./scr/reducer`
    d) `./src/actions`
    e) `./src/initState`

**Запуск**

Если необходимо заново собрать проект:

1. **npm i**
2. запуск

    2.1. **npm run build**
    2.2. **npm run start**
    2.3. **npm run dev**
